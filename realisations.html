Bijouterie Jouvenel (stage)

Dans le cadre de mon stage de deuxième année, j'ai été ammené à changer le site internet de la boutique. Pour mener à bien ce projet j'ai utilisé le framework Laravel.
Mon travail s'est porté notament sur :

  • - l'uniformisation de l'aspect graphique
  • - l'ajout de fonctionnalité pour le client

Projet de Gestion de Séjours Hospitaliers

Dans le cadre de mes études, j'ai réalisé une application web permettant de gérer les séjours des patients au sein d'un hôpital. Ce projet a été développé en utilisant le framework Symfony.
Mon travail s'est porté notamment sur :

  • - la gestion des séjours (affichage, validation des sorties, recherche par date)
  • - l'implémentation d'un système d'authentification sécurisé pour les utilisateurs
  • - l'ajout d'une gestion des utilisateurs avec des rôles spécifiques
Consulter le projet sur GitHub.

Mairie de Marcq-en-Barœul (stage)

Site Web de l'Agence Immobilière

Dans le cadre de mes études, j'ai développé un site web pour la gestion d'une agence immobilière. Ce projet utilise PHP et MySQL, et permet de gérer les biens immobiliers, les utilisateurs, et les recherches des clients.
Mon travail s'est porté notamment sur :

  • - la gestion des biens immobiliers (ajout, modification, suppression)
  • - la mise en place d'un système de recherche avancée pour les utilisateurs
  • - la génération de statistiques sur les recherches effectuées et les biens immobiliers
Consulter le projet sur GitHub.
